# Indigo's Developer Portfolio

A personal developer portfolio built with Next.js, React, Tailwind CSS, and TypeScript.
Designed with a "Space Girly-Tech" theme.

## Tech Stack
- **Framework:** Next.js 14+ (App Router)
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **Deployment:** Vercel
- **Forms:** Formspree

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
